Format data dosen untuk import (Data Jadwal Dosen SI):

Atribut wajib:
1. kode_dosen
2. nama_mata_kuliah
3. ketersediaan_waktu
4. beban_mengajar

Contoh:
kode_dosen,nama_mata_kuliah,ketersediaan_waktu,beban_mengajar
DSN-001,Analisis dan Perancangan SI,"Senin 08:00-10:00; Rabu 13:00-15:00",3
DSN-002,Basis Data,"Selasa 10:15-12:15; Kamis 08:00-10:00",3

Catatan:
- kode_dosen bersifat anonim.
- ketersediaan_waktu dapat berupa beberapa slot dipisahkan ';'.
- beban_mengajar diisi jumlah SKS.
